## 🚀 快速启动

### 1. 进入前端目录
```bash
cd web-frontend
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 访问应用
打开浏览器访问：http://localhost:8000

## 🔧 自定义配置

### 修改后端API地址
编辑 `.umirc.ts` 文件中的proxy配置：

```typescript
proxy: {
  '/api': {
    target: 'http://your-backend-url:8080',  // 修改为您的后端地址
    changeOrigin: true,
  },
}
```

### 修改主题色彩
编辑 `config/defaultSettings.ts`：

```typescript
const Settings = {
  colorPrimary: '#1890ff',  // 修改主色调
  navTheme: 'light',        // 导航主题
  // ...
};
```
